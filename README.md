# Malaga Tech Events

A simple and fast agenda to find tech events in Málaga. 

[**Live Demo**](https://malaga-tech-events.vercel.app/)

I built this personal project with **React 19**, **TypeScript** and **Vite**. It uses **Tailwind CSS** for the UI and **Leaflet** for the interactive maps.

**Note:** I used React for this version, but if I decided to take this app to production, I would consider to migrate it to **Next.js** for SEO and performance. Also, I used **mock data** to manage all the events, venues and hosts data without a backend.

Finally, I deployed the app on **Vercel**. You can see the live demo [here](https://malaga-tech-events.vercel.app/).
